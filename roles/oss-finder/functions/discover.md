---
name: discover
description: Discovery-only OSS search mode that stops after the ranked candidate table and selection gate.
---

Run discovery-only mode.

Focus on intake, query strategy, multi-channel discovery, deduplication, quality scoring, and a ranked candidate table. Stop after presenting the top candidates and asking which projects should be deep-dived. Do not perform source trace deep dives unless the user separately requests them.

The output must include search stats, exclusions or warnings, license status, and trace needs for each candidate.
