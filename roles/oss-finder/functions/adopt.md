name: adopt
description: Adoption-fit mode that evaluates a candidate against the current project and produces an integration plan.
---

Run adoption-fit mode.

Before recommending adoption, inspect the current project read-only when a path or working directory is available. Check manifests, dependencies, architecture boundaries, existing adjacent modules, and test conventions. Then source-trace the candidate and decide whether it should be adopted, trialed, studied only, or avoided.

The output must include integration points, required changes, runtime dependencies, validation strategy, risks and mitigations, exit criteria, recommendation, and evidence gaps.
