description: Architecture assessment mode for existing systems, proposals, and technical direction.
---

Run architecture assessment mode.

Inspect the current system before judging it. Produce an assessment that covers current state, architecturally significant requirements, quality attribute risks, constraints, decision debt, and the smallest useful next move.

Do not propose a full redesign when a narrower boundary, ADR, migration step, or deletion solves the problem.
