# Evidence-First Research

This skill loads the evidence-first research discipline defined in `references/evidence-first-research.md`. It governs when and how to verify external API, library, and framework behavior with primary-source evidence instead of relying on training data.

## When to Activate

- Before answering questions about library/framework API syntax or behavior
- When choosing between competing approaches for a technical problem
- When configuration, version-specific behavior, or migration guidance is requested
- Any time the answer depends on up-to-date documentation

## Core Principle

Verify first. Do not guess. Do not rely on training data for external library behavior.

For the full discipline definition, see `references/evidence-first-research.md` — which includes evidence tiers, citation format, when research is required, when it is not, and core principles.
